# Moolank 100

Canonical knowledge and transformation database for the Moolank app.

## Product idea

Moolank describes natural patterns, Bhagyank describes life direction, and the Sadhana Path turns both—together with questionnaire responses—into a practical transformation plan.

## Repository role

This repository stores structured, app-ready knowledge only. The application code remains separate.

## Core layers

1. **Moolank 1–9** — natural personality and behavioural patterns.
2. **Bhagyank 1–9** — long-term direction, lessons and growth themes.
3. **81 combinations** — interaction between Moolank and Bhagyank.
4. **Questionnaire system** — current lived expression and priority areas.
5. **Sadhana library** — safe, practical transformation practices.
6. **Transformation rules** — transparent rules for assembling a personalised path.

## Ground rules

- Traditional numerology and scientific evidence must remain separate.
- Numerology claims must not be presented as scientific facts.
- Medical, psychiatric, diagnostic, criminal, addiction and fixed negative labels are prohibited.
- Questionnaire results describe development areas, not diagnoses.
- Every recommendation must be practical, proportionate and revisable.
- GitHub is the permanent source of truth.
